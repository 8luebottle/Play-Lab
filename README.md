## Naming Convention

### File
* snake case    
* all lowercase    
**```<directory_name>_<file_name>```**
  * e.g. ```channel_print_integers```
  
### Commit Message
* start w/ date 
* end w/ subject   
**```MM.DD.YYYY : Subject```**
  * e.g. ```07.10.2020 : Go```
